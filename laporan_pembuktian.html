Tahap 2: Normalisasi Matriks ($R$)

Rumus & Implementasi Angka: $$ r_{ij} = \frac{x_{ij}}{\sum x_{ij}} $$ Contoh (Baris 1, Kolom 1):
r11 = 4 / 11 = 0.3636

Tahap 3: Normalisasi Terbobot ($D$)

Rumus & Implementasi Angka: $$ y_{ij} = r_{ij} \times w_j $$ Bobot Kriteria ($w_j$) = [ 0.4, 0.3, 0.15, 0.15 ]

Contoh (Baris 1, Kolom 1):
y11 = 0.3636 × 0.4 = 0.1455
Alt C1. Jenis Keluhan C2. Harga C3. Jarak Tempuh C4. Usia Kendaraan
Paket Standar 0.1455 0.1000 0.0450 0.0450
Paket Servis Bulanan 0.1091 0.1000 0.0600 0.0600
Paket Servis Lengkap 0.1455 0.1000 0.0450 0.0450

Tahap 4: Nilai Max ($S_+$) & Min ($S_-$)

Implementasi Angka (Alternatif 1 - Paket Standar): $$ S_+ = \sum \text{Benefit} \quad ; \quad S_- = \sum \text{Cost} $$ S+ = (Kolom C1 + C2) = 0.1455 + 0.1000 = 0.2455
S- = (Kolom C3 + C4) = 0.0450 + 0.0450 = 0.0900
Alternatif Benefit ($S_+$) Cost ($S_-$)
Paket Standar 0.2455 0.0900
Paket Servis Lengkap 0.2455 0.0900
Paket Servis Bulanan 0.2091 0.1200
TOTAL ($S_-$) - 0.3000

Tahap 5: Bobot Relatif ($Q_i$)

Rumus Lengkap: $$ Q_i = S_{+i} + \frac{\sum S_{-}}{S_{-i} \times \sum \frac{1}{S_{-}}} $$ Data Global: • $\sum S_-$ = 0.3000
• $\sum \frac{1}{S_-}$ = 30.5556

Implementasi (Alternatif 1):
Q1 = 0.2455 + ( 0.3000 / ( 0.0900 × 30.5556 ) )
Q1 = 0.3545
Alternatif Bobot Relatif ($Q_i$)
Paket Standar 0.3545
Paket Servis Lengkap 0.3545
Paket Servis Bulanan 0.2909

Tahap 6: Utilitas ($U_i$) & Perankingan

Implementasi Angka: $$ U_i = \frac{Q_i}{Q_{max}} \times 100\% $$ Q_max = 0.3545
U1 = ( 0.3545 / 0.3545 ) × 100 = 100.00%
